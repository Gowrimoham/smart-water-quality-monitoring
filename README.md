# Smart Water Quality Monitoring System

## Overview
This project measures water turbidity using a sensor and displays water quality.

## Components Used
- Arduino Uno
- Turbidity Sensor (ET5508)
- 16x2 LCD Display
- I2C Module

## Working
- Sensor reads turbidity value
- Value converted to approximate NTU
- Displayed on LCD

## Applications
- Drinking water monitoring
- Industrial water testing
